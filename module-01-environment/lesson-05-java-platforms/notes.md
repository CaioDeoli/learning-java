# Conhecendo as plataformas Java
- O universo Java é composto por várias plataformas
- Para rodar em vários tipos de sistemas e dispositivos
- Maquina virtual + compilador + bibliotecas + algumas ferramentas
- Principais plataformas
  - Java SE (standard edition)
  - Jakarta EE (enterprise edition) - antigo java ee
  - Java ME (micro edition)
  - Java Card
- Foco desse curso Java SE. Base de todo dev Java
- As plataformas Java não são um produto final, elas possuem especificações de como a plataforma deve funcionar. É pra padronizar a tecnologia
- Mais vagas no mercado: Java SE e Java EE

## Java SE
- Java Platform, Standard Edition
- Destinada ao desenvolvimento para desktops e servidores
- Especificacao da linguagem java e apis importantes, e especificacao da VM

## Jakarta EE
- Jakarta Enterprise Edition (antigo Java EE e J2EE)
- Aplicações que rodam em servidores
- Bibliotecas e funcionalidades para implementar software Java distribuido, que suporta escalabilidade, integridade...
- Plataforma guarda chuva
  - Especificacoes
    - Jakarta Servlet
    - Jakarta Persistence (JPA)
    - Jakarta Bean Validation
    - Jakarta Context and Dependency Injection (CDI)
    - Jakarta RESTful Web Services (JAX-RS)
    - Jakarta Enterprise Beans (EJB)
- jakarta.ee
- Especificacao: especifica como funciona a tecnologia
- Implementação: é a tecnologia que implementa e especificacao

## Java ME
- Java Platform, Micro Edition
- Permite criacao de sistemas embarcados em dispositivos compactos
- Celulares, impressoras, controles remotos
- Interfaces e serguranca

## Java Card
- Permite rodar Java em cartoes inteligentes
- Cartoes de banco e chips de celular
