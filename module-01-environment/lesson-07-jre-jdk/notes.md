# JRE e JDK

## JRE
- Java Runtime Environment (JRE)
- Ambiente de Execução Java
- Composto por
  - Bibliotecas padrão: numeros, datas, colecoes, redes
  - Class Loader: carrega programas java
  - JVM: interpreta bytecode
- É o que realmente vai ser instalado nas máquinas para rodar Java
- Windows, MacOS, Linux

## JDK
- Java Development Kit
- Kit de Desenvolvimento Java
- Composto por
  - Compilador (Java -> Bytecode)
  - Outras ferramentas e utilitarios
  - JRE
- Como dev voce só precisa do JDK
- Windows, MacOS, Linux
