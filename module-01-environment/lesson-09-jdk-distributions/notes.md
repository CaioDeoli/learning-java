# Conhecendo as distribuições JDK
- JDK existe várias opções de diferentes fornecedores (as distribuições)
- opções pagas e gratuitas
- como funciona o desenvolvimento da linguagem java. JCP. Processo formal na evolução de uma tecnologia java
- JSR -> java specification request
- isso nao é a implementação, isso sao as regras e padrões. outra empresa implementa
- openjdk.net -> implementação de referência do que foi definido no JCP
- cada versão do java tem uma jsr
- JEP sao as sugestões de melhorias para o openjdk, implementar funcionalidades no openjdk. acaba virando um roadmap com algumas implementadas outras não
- JSR -> openjdk (RI) reference implementation -> distribuicoes por fornecedores
  - Oracle openjdk
  - Oracle jdk
  - Eclipse temurin <-> a que vamos usar
  - Amazon corretto
- openjdk sozinha não é um software utilizavel 
- fornecedores fazem suas próprias melhorias que podem voltar para virar um JSR para uma versão futura
- TCK -> processo para ver se a distribuição estar funcionando bem
- a que ele recomenda para o curso
- cada fornecedor pode dar suporte por tempos diferentes a certas versoes
