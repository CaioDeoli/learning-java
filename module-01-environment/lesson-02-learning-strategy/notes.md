# Como aprender Java e pedir ajuda

## Como estudar
- Foque em apenas um curso e na sequência do curso
- Pratique
- Faça os desafios sozinho mesmo que dure horas
- Tente se divertir com os problemas
- Não corra para terminar o curso
- Escreva, não copie. Digite
- Não menospreze os exemplos
- Não se assuste com a quantidade de palavras novas
- Siga uma rotina de estudos

## Como fazer perguntas
  - Saiba fazer perguntas
  - Tente resolver sozinho antes de perguntar
  - Escreva um título que descreva a dúvida de forma clara
    - x "Erro no código da aula"
    - v "illegalArgumentException lançada ao chamar método ClienteService.salvar"
  - Contextualize e descreva como reproduzir o problema
    - Como você chegou no problema
    - O que você já testou
    - Como reproduzir o erro
    - Releia a pergunta e confira se está fácil de entender
  - Envie os trechos de códigos relevantes
  - Evite enviar links de repositório
  - Envie prints de telas relevantes
