# Conhecendo e usando convenções de código
- Trabalhar com uma equipe que seguem todos os mesmos padrões em um mesmo projeto é muito importante
- Padrões e boas práticas. Convenções de código (code convencions)
- As que existem são muito semelhantes
- Mostra profissionalismo
- Reduz custo de manutenção
- Melhora a legibilidade

## Exemplos
- Espaço antes das chaves
- Abertuira das chaves na mesma linha da declaração da classe ou método
- Identação padronizada (geralmente são 4)
- Nome da classe/programa é o mesmo do nome do arquivo, com as iniciais maiusculas em camelcase
- 2 convenções
  - oracle.com - Code Conventions for the Java
  - google.com - Google Java Style Guide
