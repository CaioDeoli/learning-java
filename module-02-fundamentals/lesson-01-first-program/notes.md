# Criando o primeiro programa Java

- Estrutura básica do Java
- Java é case sensitive
- Padrão de usar nome de classe em camel case e começando com letra maiuscula
- identação são os espaços para hierarquizar os blocos pais e filhos
- Sem identação funciona, mas não é recomendado
- Nome da classe é o mesmo nome do arquiv
- Extensão .java

```java
// Declaração de "programa" (classe)
public class OlaMergulhador { // Abrindo bloco de código

  // Ponto/função inicial de um programa (main)
  public static void main(String[] args) {
    /*
    System -> classe
    out -> na saida
    println() -> método de impressão
    () -> aonde coloca os argumentos
    "" -> string/texto
    ; -> sempre termina com ";"
    */
    System.out.println("Olá, mergulhador!");
  }

}
```
