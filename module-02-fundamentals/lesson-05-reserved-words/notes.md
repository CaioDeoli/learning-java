# Palavras reservadas/chave
- Palavras especiais reservadas que tem significado para o compilador, para interpretar o que seu código fonte esta tentando fazer. Elas tem um significado especial
- Não usar como nome de métodos, classes, variaveis
- Lista
  - abstract
  - boolean
  - break
  - byte
  - case
  - catch
  - char
  - class
  - const * -> sem utilidade mas reservada
  - continue
  - default
  - do
  - double
  - else
  - extends
  - final
  - finally
  - float
  - for
  - goto * -> sem utilidade mas reservada
  - if
  - implements
  - import
  - instanceof
  - int
  - interface
  - long
  - native
  - new
  - package
  - private
  - protected
  - public
  - return
  - short
  - static
  - strictfp
  - super
  - switch
  - synchronized
  - this
  - throw
  - throws
  - transient
  - try
  - void
  - volatile
  - while
  - assert
  - enum
- Tem uma semantica as palavras, significado especial
```md
**public class** OlaMergulhador {

  **public static void** main(String[] args) {
    System.out.println("Olá, mergulhador");
  }

}
```
- se usar uma palavra reservada no nome de uma classe vai dar erro
