# Tipos primitivos - float e double

- Únicos tipos que são capazes de armazenar números com pontos flutuantes
- float -> pressição de 6 casas decimais
- double -> pressição de 15 casas decimais

[DoubleExample.java](DoubleExample.java)
[FloatExample.java](FloatExample.java)

- Os tipos double e float não devem ser usados para armazenar valores que precisam de muita precisão. Exemplo: valores monetários.
- Para tipos de situação vamos utilizar a classe BigDecimal, que é mais seguro
